# 🧠 Mind–Body Map Distilled

### "뇌가 마음을 만든다" 고 **말하는 것** 과,

$$\text{물리 사실}\;\lbrace P\rbrace \;\xrightarrow{?}\; \text{경험 사실}\;\lbrace Q\rbrace \qquad(\text{화살이 완전히 닿는가?})$$

### 마음과 물질의 관계가 **왜** 2천 년째 안 풀리는지 — 이원론·물리주의·기능주의가

### **각각 어디서 성공하고 어디서 무너지는지** 를 직접 추적하는 것은 **다르다.**

<br/>

> *좀비 논변을 "철학적 상상" 으로 **넘기는 것** 과,*
>
> $$P_1:\ \text{좀비 개념가능} \;\Rightarrow\; P_2:\ \text{개념가능}\to\text{형이상학적 가능} \;\Rightarrow\; C:\ \text{물리주의 거짓}$$
>
> *그것이 **물리주의에 대한 정밀한 압박** 이고 — 어느 전제(P₁? P₂?)를 부정해야 반박되는지, 왜 아직 아무도 결판내지 못했는지를 아는 것은 다르다.*
>
> *"메리의 방"(색을 모르고 태어난 색채 신경과학자)을 퀴즈로 **아는 것** 과,*
>
> $$\text{메리가 물리적 사실을 모두 안다} \;\wedge\; \text{방을 나와 새것을 안다} \;\Rightarrow\; \exists\,\text{비물리적 사실}$$
>
> *그것이 **설명적 간극의 가장 날카로운 도구** 이고, 물리주의를 정확히 어디서 겨냥하는지를 파악하는 것은 다르다.*
>
> *"마음 = 소프트웨어" 라는 기능주의를 **인용하는 것** 과,*
>
> $$\text{기능적 역할 동일} \;\Rightarrow\; \text{마음 동일}\ ? \qquad(\text{다중 실현} \;\checkmark\;|\; \text{감각질}\;?)$$
>
> *다중 실현이라는 성공과, 경험의 **내용**(what it is like)이 기능 기술 밖으로 빠져나가는 한계가 어디서 갈리는지를 아는 것은 다르다.*
>
> *설명적 간극을 "뇌과학이 발전하면 풀린다" 며 **낙관하는 것** 과,*
>
> $$\text{"C섬유가 발화한다"} \;\xrightarrow{?}\; \text{"고통스럽다"} \qquad(\text{왜 전자가 후자를 설명 하는가?})$$
>
> *환원(Dennett)·비환원(Chalmers)·신비주의(McGinn)가 각각 **어떤 비용** 을 치르고 간극을 다루는지를 아는 것은 다르다.*

<br/>

**뇌는 마음의 *상관물* 이지 *설명* 이 아니다** — 라플라스식 신경과학이 "C섬유 발화 = 고통" 을 완성해도, *왜 그것이 고통으로 느껴지는가* 라는 화살이 남는다. 이 레포는 그 화살이 끊기는 자리에 정확히 깃발을 꽂는다.

<br/>

**다루는 입장과 도구 (지형에서 종합으로)**

데카르트적 의심 · 쉬운 문제 / 어려운 문제 · 범주 오류 · 설명 층위(Marr) · 실체 이원론 · res cogitans / res extensa · 상호작용 문제 · 속성 이원론 · 수반(supervenience) · 동일론 · 비환원적 물리주의 · 좀비 논변 · 메리의 방(지식 논변) · 역전된 감각질 · 현상 개념 전략 · 기능주의 · 다중 실현 가능성 · 튜링 테스트 · 중국어 방 · qualia · 범심론 · 결합 문제 · 중립적 일원론 · Russell·James 계보 · 설명적 간극(Levine) · 환원/비환원/신비주의 전략 · 반증 가능한 예측 비교 · 3인칭 + 1인칭 + 계산 모델의 삼각측량

## 🎯 이 레포에 대하여

마음을 다루는 자료는 대부분 **"뇌가 마음을 만든다"** 또는 **"의식은 결국 신경의 부산물"** 에서 멈춥니다. 하지만 그 설명이 *왜* 아직 불완전한지(설명적 간극), 좀비 논변과 메리의 방이 *어떻게* 물리주의를 정밀하게 압박하는지, 기능주의의 "마음 = 소프트웨어" 가 *어디서* 빠져나가는지(qualia), 범심론이 *무엇을* 해결하고 *무엇을* 새로 만드는지(결합 문제) — 이런 "왜" 와 "어디서" 는 제대로 그려지지 않습니다.

이 레포는 **마음-몸 문제를 입장들의 지형도** 로 환원합니다. 각 입장에 **핵심 전제 · 논증 단계 · 압박하는 사고실험 · 설명적 간극의 위치 · 반증 가능한 예측** 을 명시하고, 어떤 입장이 어떤 트레이드오프를 치르는지 표와 다이어그램으로 보입니다. "틀렸다/맞다" 가 아니라 *무엇을 설명하고 어디서 멈추는가* 로 배치합니다.

| 일반 자료 | 이 레포 |
|----------|---------|
| "뇌가 마음을 만든다" | **상관 ≠ 설명** — 신경 상관물(NCC)이 완성돼도 *왜* 그것이 경험으로 *느껴지는가* 가 남는다. 설명의 화살 $\{P\}\xrightarrow{?}\{Q\}$ 가 닿는지가 문제. **경계**: 쉬운 문제는 풀림, 어려운 문제는? $\square$ |
| "의식은 그냥 복잡한 계산" | **기능 ≠ 경험** — 기능주의는 다중 실현을 설명($\checkmark$)하지만, 좀비 논변이 *기능 동일 + 경험 부재* 의 개념가능성을 들이댄다. qualia가 기능 기술 밖으로 빠짐 $\square$ |
| "좀비 논변은 공상" | **정밀한 압박** — $P_1$(개념가능) → $P_2$(개념가능→형이상학적 가능) → $C$(물리주의 거짓). 반박하려면 $P_1$ 또는 $P_2$ 를 부정해야 하고, 둘 다 비용이 크다 $\square$ |
| "메리의 방은 말장난" | **지식 논변** — 물리 사실을 다 아는 메리가 방을 나와 *새것* 을 안다면, 그 새것은 비물리적. 표상주의 반론(능력 지식)과의 정확한 충돌점 $\square$ |
| "이원론은 비과학" | **직관의 힘** — 상호작용 문제(비물질이 어떻게 뇌에?)는 치명적이지만, "나는 *이것* 을 경험한다" 는 직관은 제거하기 어렵다. 속성 이원론이 왜 살아남는가 $\square$ |
| "범심론은 신비주의" | **결합 문제** — 경험을 기초로 두면 간극은 사라지지만(매력), 미시 경험이 *어떻게* 통합된 하나가 되는가(비용). 트레이드오프를 정직하게 $\square$ |
| "간극은 곧 메워진다" | **간극의 좌표** — Levine의 "C섬유 발화 = 고통" 이 *왜* 설명이 아닌가. 환원(Dennett)·비환원(Chalmers)·신비주의(McGinn) 각 전략의 비용 표시 $\square$ |
| "뉴런이 곧 마음" | **범주 오류** — 두 기술 체계(물리적/현상적)의 혼동. Marr의 3층위로 "층위 혼동" 을 진단(→ levels-of-explanation) $\square$ |
| 입장 나열 | matplotlib·Mermaid 로 **입장 비교 레이더 차트** · **좀비/메리의 방 논변 흐름도** · **설명 층위 매핑(L0~L6)** · **각 입장의 반증 예측 비교표** 를 직접 그려 지형을 *눈* 으로 본다 |
